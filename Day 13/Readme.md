# What is Boto3?
Boto3 is the official AWS SDK (Software Development Kit) for Python. It allows developers to interact with Amazon Web Services (AWS) like S3, EC2, DynamoDB, Lambda, IAM, and many more using Python.

#  Why Use Boto3?
✅ Automate AWS services with Python
✅ Manage AWS resources programmatically
✅ Perform operations like uploading files, creating EC2 instances, and managing databases

# Installing Boto3
pip install boto3
